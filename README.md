# Binary Tree Implementation in C

## Project Overview
This project demonstrates the implementation of the Binary Tree data structure using the C programming language. A Binary Tree is a hierarchical data structure where each node contains data and pointers to its left and right child nodes.

## Features
- Create a Binary Tree
- Insert nodes into the tree
- Search for an element
- Preorder Traversal
- Inorder Traversal
- Postorder Traversal
- Display tree elements

## Technologies Used
- C Programming
- Data Structures and Algorithms (DSA)

## Traversal Techniques

### Preorder Traversal
```text
Root → Left → Right
```

### Inorder Traversal
```text
Left → Root → Right
```

### Postorder Traversal
```text
Left → Right → Root
```
